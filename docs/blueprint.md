## Core Features:

- Homepage Hero Section: Visually appealing hero section with key stats and clear CTAs to drive user engagement.
- Guides Listing: Display comprehensive guides in a clear, card-based layout for easy access.
- Interactive Gallery: Showcase devices with a lightbox feature, allowing users to explore high-quality images.
- Product Catalog: Organize recommended accessories with detailed specifications and affiliate links.
- Contact Form: Implement a functional contact form with EmailJS integration for user inquiries.
- Interactive Map: Integrate Geoapify Maps to display a location; map becomes interactive on click.
- Floating Buttons: Sticky buttons for WhatsApp and social media to be placed in the right hand bottom corner

## Style Guidelines:

- Primary color: Deep blue (#2563eb) to convey technology and trust.
- Secondary color: Emerald green (#10b981) for innovation and success.
- Background color: Very light blue (#F0F9FF) to ensure high readability.
- Font: 'Inter' (sans-serif) for a clear and modern reading experience. Note: currently only Google Fonts are supported.
- Use a modular architecture based on features (e.g., `features/home`, `features/services`) to keep components well organized.
- Employ Framer Motion for subtle animations, like counting up for statistics in the hero section, and entry/exit transitions for the gallery lightbox.
- Utilize React Icons for consistent and recognizable icons across the site.
